Premiere utilisation - configurez votre projet Firebase

Creez un projet Firebase gratuit puis copiez les valeurs depuis Parametres du projet > Configuration du SDK.

Guide pas-a-pas
  1. Aller sur console.firebase.google.com
  2. Creer un projet (nom libre, Analytics optionnel)
  3. Ajouter une application Web
  4. Copier les valeurs de firebaseConfig
  5. Activer Authentication > Google
  6. Creer une base Firestore Database
